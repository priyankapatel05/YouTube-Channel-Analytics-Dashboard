# 📊 YouTube Channel Analytics Dashboard

## 🚀 Project Overview
This project presents an end-to-end analysis of my YouTube channel data using PostgreSQL, Power BI, and DAX. The objective was to analyze channel performance, identify growth patterns, evaluate content effectiveness, and generate actionable insights from real-world data.

## 📅 Analysis Period
**June 2025 – June 2026**

## 📈 Dataset Summary
- 442 YouTube Videos Analyzed
- 335K+ Total Views
- 194 New Subscribers
- 89K Impressions
- Real-world data from my own YouTube channel

## 🛠️ Tools & Technologies
- **PostgreSQL (PgAdmin)** – Data Storage & SQL Analysis
- **SQL** – Data Extraction & Insights
- **Power BI** – Interactive Dashboard Development
- **DAX** – Measures & Calculated Columns

## 📊 Dashboard Pages

### 1️⃣ Channel Overview
Provides a high-level summary of channel performance including:
- Total Views
- Watch Hours
- Subscribers Gained
- Impressions
- CTR Analysis
- Monthly View Trends

### 2️⃣ Video Performance Analysis
Analyzes individual video performance through:
- Top Videos by Views
- Top Videos by Watch Time
- CTR Performance Analysis
- Zero-View Video Identification

### 3️⃣ Growth & Content Strategy
Focuses on growth opportunities and content optimization:
- Peak Growth Months
- Content Performance Trends
- Best Publishing Days
- SQL-Driven Insights
- Content Gap Analysis

## 🔍 Key Insights
- August 2025 recorded the highest growth with **50K+ views**
- Top-performing video generated **30K+ views**
- Identified **104 videos with zero views** 
- Friday showed the highest average engagement and views
- Content gap observed during Dec 2025 – Jan 2026

## 🎯 Business Impact
This analysis helps content creators:
- Identify high-performing content patterns
- Optimize posting schedules
- Improve content strategy
- Monitor channel growth effectively
- Detect underperforming content
